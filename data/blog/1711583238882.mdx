目前来说，谷歌的 Gemini 是免费使用的，包括它的 API 也是。

### 直接使用

首先，我们访问它的[官网](https://deepmind.google/technologies/gemini/#introduction)并登录谷歌账号。

> 需要科学上网 - 国内不支持

![](https://note.ihsxu.com/api/imgs/1711583315302.webp)

然后进入到对话页面开始进行对话，比如让它介绍一下自己：

![](https://note.ihsxu.com/api/imgs/1711583450317.webp)

它还支持：

*   **上传图片并对图片中的内容进行解析**
*   **语音对话（又可以练英语了）**

![](https://note.ihsxu.com/api/imgs/1711771512158.webp)

比较遗憾的是它现在还无法支持画图功能。

### 免费 API Key

*   登录到 [Google AI Studio](https://aistudio.google.com/app/apikey) 网站

![](https://note.ihsxu.com/api/imgs/1711640972089.webp)

*   创建 API Key

![](https://note.ihsxu.com/api/imgs/1711641007796.webp)

选择「Create API Key in new project」来创建一个新 key 就可以啦。

比如我们可以在 ChatBox 中进行使用：

![](https://note.ihsxu.com/api/imgs/1711773555588.webp)

![](https://note.ihsxu.com/api/imgs/1711773679304.webp)
