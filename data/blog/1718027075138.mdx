不知道有没有小伙伴发现，最近几天在使用 docker 拉取官方镜像的时候经常会失败，三金开始还以为镜像作者是不是删镜像了😂，后面登录 dockerhub 之后发现镜像还在的，只不过由于一些原因导致我们无法在国内正常进行拉取而已。

而在 6 月 6 日，上海交大的 Docker Hub 镜像加速器也宣布因收到通知要求被下架，请用户及时进行切换。

![](https://note.ihsxu.com/api/imgs/1718029210681.webp)

**不止上海交大的镜像源，国内镜像源好多都停止服务**，比如:

*   Docker 中国官方镜像站
*   DaoCloud 镜像站
*   七牛云镜像站
*   网易云镜像站
*   还有阿里、腾讯、百度、南京大学以及中科大等国内大家熟知的镜像站也都无法使用

就目前来说，**Github CDN 镜像还可以正常使用**，但是据传后续未经审查通过的镜像服务器都会陆续被进行限制，无法提供服务。

小伙伴们有正在使用国内镜像站但却无法正常拉取镜像的，可以尝试切换到 Github CDN 镜像上，且用且珍惜😭
