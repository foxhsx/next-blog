三金玩儿过很多博客平台，刚开始是 CSDN，后面慢慢转移到了掘金，再后来又搞个人站（Hexo、VuePress、VitePress），最后干脆把公众号、知乎啥的也开通了。整到后面就是发一篇文章，然后再挨个儿手动复制到其他平台上，很是麻烦。

你说是想做自媒体吧，又不是专业的，但是就是想玩一玩儿，不过也没坚持太久，慢慢的就放弃了 CSDN 和知乎以及公众号（其实还有慕课、简书😂），因为太费精力了。

后来逛 Github 的时候偶然间发现了 [Wechatsync](https://github.com/wechatsync/Wechatsync)，它的项目描述是这样的：

> **一键同步文章到多个内容平台，支持今日头条、WordPress、知乎、简书、掘金、CSDN、typecho各大平台，一次发布，多平台同步发布。解放个人生产力**

瞧瞧，这不正是我想要的嘛！！

那如何使用呢？

### 安装

首先，这款产品是一个浏览器插件，对于谷歌浏览器来说，一般我们可以到 chrome 应用商店进行下载安装。不过由于众所周知的原因，我们无法打开商店的页面，所以作者也贴心的在该产品的[官网](https://www.wechatsync.com/?utm_source=github_info)上直接给出了下载链接。

![](https://note.ihsxu.com/api/imgs/1719153732166.webp)

可以看到除了安装浏览器插件之外，还有**在线 Markdown 编辑器**的功能，点开它之后也可以在这个页面写文章然后同步到社交平台上：

![](https://note.ihsxu.com/api/imgs/1719153910612.webp)

回到安装插件，根据官网的教程安装好插件之后我们就可以开始使用了。

### 使用

在使用之前，我们**需要选择同步的平台进行登录**，只有登录成功之后，插件才能识别到这些平台（可能会有一分钟的延迟）。

使用时分两个场景：直接多平台同步和公众号同步到其他平台。

#### 多平台同步

打开想要同步的文章，然后**鼠标右击浏览器，在弹出的菜单中选择「提取文章并同步」**：

![](https://note.ihsxu.com/api/imgs/1719154638442.webp)

然后在**新弹出来的页面中点击右上角的「同步」按钮，选择自己要同步的平台**即可：

![](https://note.ihsxu.com/api/imgs/1719154922672.webp)

#### 公众号同步到其他平台

**打开一篇公众号文章就可以看到左上角的「同步该文章」按钮，点击它就可以进行同步了**：

![](https://note.ihsxu.com/api/imgs/1719155478985.webp)

### 发布

同步号之后，**并不是直接就会自动发布**，还是**需要通过同步完成后提供的跳转链接到对应平台的草稿箱中去自行进行发布**，这里只要轻抬小手点击发布就可以啦～

真的方便又快捷，感兴趣的小伙伴快去试试吧。
