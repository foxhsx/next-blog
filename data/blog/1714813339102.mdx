对于程序猿和有过建站经验的小伙伴来说，Cloudflare 想必大家都不陌生，它是一家提供内容分发网络（CDN）、互联网安全性、抗 DDoS（分布式拒绝服务）和分布式 DNS 服务的美国公司。

而让大家对它比较中意的一点是：它那广为人知的慷慨免费套餐，可以让一些中小型网站几乎能以零成本运行。

Cloudflare 提供了以下一些常见的免费服务:

1.  **DNS 管理**: Cloudflare 提供免费的 DNS 管理服务，可以帮助用户管理域名的 DNS 解析。

2\. **CDN**: Cloudflare 提供免费的内容分发网络（CDN）服务，可以加速用户网站的内容传输。

3\. **SSL/HTTPS**: Cloudflare 可以为用户的网站提供免费的` SSL/HTTPS` 证书，确保网站访问的安全性。

4\. **DDoS 防护**: Cloudflare 的免费服务包括基本的 DDoS 攻击防护功能，可以帮助用户的网站抵御一些常见的 DDoS 攻击。

5\. **分析报告**: Cloudflare 提供免费的网站分析报告，可以帮助用户了解网站的访问情况。

6\. **工具包**: Cloudflare 提供了一些免费的开发者工具，如 Cloudflare Workers、Cloudflare Tunnel 等。

7\. **缓存优化**: Cloudflare 的免费服务包括基本的缓存优化功能，可以加快网站的响应速度。

而在开源大模型开始互卷的时候，它又及时地提供了免费使用开源大模型的演练场，让更多的人能够接触到这些最新的 AI 模型，降低了使用门槛，接下来就让我们一起来看看吧。

![](https://note.ihsxu.com/api/imgs/1714820488366.webp)

页面由三部分组成：

1.  左侧的配置区域：可以选择大模型类型、系统消息以及 token
2.  右侧是主体区域，每个 User 就是一个新的对话，在 Enter a message 输入框中输入文字，点右下角的 Run 按钮开始对话；也可以点击 Add a message 创建一个新的对话
3.  右下角除了 Run 按钮之外，还有一个 Clear 按钮，用来清空所有对话

这个 Playground 提供的开源大模型相比 LLSYS 来说少那么几个，目前一共有 30 个（不过 LLSYS 也不仅只提供了开源大模型🤔）。
