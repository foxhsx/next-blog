小伙伴们大家好，我是三金～

今天在逛 AI 新闻的时候，发现 Mistral AI 还有个免费计划，好奇心驱使之下去他们家网站研究了一下，发现即使不绑定付费方式、只需要手机验证一下就能参与。最最最可以的是，国内手机也可以参与，且国内也能直连。

接下来让我们一起看下如何开通 Mistral 的免费计划～

首先我们访问 Mistral 的[控制台](https://console.mistral.ai/)，如果你还没有账号的话需要先注册一个账号：

*   可以选择 Google 账号
*   也可以选择使用 Microsoft 账号
*   也可以通过电子邮箱注册

![](https://note.ihsxu.com/api/imgs/1727618179074.webp)

注册好之后登录，我们就可以看到 Mistral 的控制台，点击左侧菜单的 **Billing** 即可看到它的免费计划：

![](https://note.ihsxu.com/api/imgs/1727507377076.webp)

开通时只需要手机接收验证码即可，国内手机也可以使用，我就是使用国内手机进行开通的。

接下来我们再到 API Keys 页面创建一个 API Key：

![](https://note.ihsxu.com/api/imgs/1727618534908.webp)

注意！注意！注意！创建好的 key 需要粘贴复制下来，保存在本地，因为关闭之后就不会再进行展示了！

有了 API Key 之后我们就可以进行使用了，不过在使用前记得配置 Mistral API 的[服务地址](https://api.mistral.ai/v1/chat/completions)。

### Chatbox 测试

在 Chatbox 的设置中，我们将 Mistral 的 API 域名、API 路径、密钥以及模型名称都填入到对应的输入框中，然后点击保存。

![](https://note.ihsxu.com/api/imgs/1727619177833.webp)

在 Just Chat 中选择我们刚刚保存的 Mistral 模型即可开始对话了：

![](https://note.ihsxu.com/api/imgs/1727619371847.webp)

不过模型的智商好像不太行，对于 9.11 和 9.9 哪个大的问题它答不对：

![](https://note.ihsxu.com/api/imgs/1727619617017.webp)

不过呢，我也不指望用它来做数学题，只要能完成一些设定好的指令就行。

接下来把它集成到 Dify 中去。

### 在 Dify 中使用

在 Dify 中我们打开设置，并定位到模型供应商中找到 Mistral，只需要填入 API Key 即可，Mistral 的 API 域名 Dify 已经帮我们配置好了，所以不需要过多关心：

![](https://note.ihsxu.com/api/imgs/1727621333985.webp)

![](https://note.ihsxu.com/api/imgs/1727508466805.webp)

在 dify 中一共支持以下 6 种 mistral 模型：

*   open-mistral-7b：23 年 9 月发布的模型，官方博客当时主要是对标 LlaMA 2 13B 和 Llama1 34B；
*   open-mixtral-8x7b：23 年 12 月发布，官博内容介绍对标 Llama2 70B；
*   open-mixtral-8x22b：官方称迄今为止对好的开源模型，与 24 年 4 月发布。在编码和数学任务中表现最佳（测试答不对 9.9 和 9.11 比大小）；
*   mistral-small-latest：与 24 年 9 月发布的最新的企业级小型模型
*   mistral-medium-latest：Mistral Medium 很快就会被弃用；
*   mistral-large-latest：与 24 年 7 月发布的用于高复杂性任务的顶级推理模型（测试答不对 9.9 和 9.11 比大小）。

我们创建一个测试用的空白应用来测试一下是否能正常调通：

![](https://note.ihsxu.com/api/imgs/1727508682129.webp)

可以正常调通，我们不需要 Mistral 做很复杂的计算任务，对于 Dify 来说，大多数情况下我们只需要它处理一些固定性地、流水线式的任务即可。
