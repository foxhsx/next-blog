*   如何免费获取 ChatGPT API Key？
*   使用 Docker 一键免费部署 63.8k 的私人 ChatGPT 网页应用

期间有一些读者朋友在后台私信我，使用 Github 账号领取了 API Key，但是不会使用 Docker，没有相关经验，有没有更便捷的一些工具推荐？

😏今天它来了——在 Github 上拥有 17.6K 的 ChatBox 项目，它是 ChatGPT、Claude 和其他 LLMs 的桌面客户端，支持在 Windows、Mac、Linux、ios 和 Android 上使用。

[Github 地址](https://github.com/Bin-Huang/chatbox)

[官网地址](https://chatboxai.app/zh)

![](https://note.ihsxu.com/api/imgs/1710558759551.webp)

### 下载

我们通过官网提供的[下载页面](https://chatboxai.app/zh#download)选择适合的操作系统进行下载：

![](https://note.ihsxu.com/api/imgs/1710558924732.webp)

下载安装完成之后，我们进入到设置中：

*   选择 AI 供应商：比如 OpenAI API
*   设置 API 密钥：这里填入 API Key
*   设置代理地址：填入 GPT\_API\_free 中提供的代理地址

![](https://note.ihsxu.com/api/imgs/1710559247781.webp)

保存之后，我们就可以开始对话了：

![](https://note.ihsxu.com/api/imgs/1710559402691.webp)

除了基础的对话之外，它提供了几个预设角色，分别是：

*   Image Creator (Example) - 生成图片（如果使用的是免费的 API Key，则不支持，因为模型不一样）
*   Just chat - 聊天对话窗口
*   Translator (Example) - 翻译工作者，将输入的内容都转化成英文；
*   Software Developer (Example) - 软件开发人员，预设使用 Golang 和 Angular 开发安全应用程序的架构和代码；
*   Social Media Influencer (Example) - 社交媒体工作者，比如根据一个主题生成一个朋友圈文案或者微博文案；
*   Travel Guide (Example) - 旅游指南，比如根据目的地和旅行天数为你生成一份行程表；
*   Markdown 101 (Example) - 根据你的需求，为你生成 Markdown 语法的内容；
*   翻译助手 - 将中文翻译成英文，将非中文的翻译成中文
*   夸夸机 - 一个全能鼓励师
*   小红书文案生成器
